# Product Guide - Muse

## Initial Concept
Text to Speech

## Target Users
- Individuals looking for an accessible way to consume written content (e.g., people with visual impairments or reading difficulties).
- Content creators who need high-quality voiceovers for videos or podcasts.

## Project Goals
- Provide a seamless and natural-sounding Text-to-Speech experience using advanced AI models like ElevenLabs.
- Offer a cross-platform solution (Android and iOS) for consistent access to TTS features.
- Support multiple languages and customizable voice profiles to cater to a global audience.

## Key Features
- Direct integration with the ElevenLabs API for high-fidelity voice generation.
- **Audio Isolation:** Remove background noise from audio files using ElevenLabs' advanced models.
- **Audio Export and Sharing:** Export generated TTS audio to MP3 format and share it via system sheets.

## User Experience Priorities
- Simplicity and speed: A "one-tap" approach to start converting text to speech.
